## Auth Flow (Closed Loop)

1. `App.tsx` → `fetchHealth()` (public, no token)
2. `health_handler` returns `auth_enabled: bool`
3. If true → `LoginScreen` renders with `mnk_...` input
4. Token validated via `apiFetch('/memories?limit=1', token)` → BLAKE3 check
5. On success → dashboard with token in Preact state only (no localStorage)
6. All tabs pass `token` prop → all `apiFetch` calls include `Authorization: Bearer`
7. Any 401/403 → `UnauthorizedError` → `onUnauthorized()` → back to login gate
