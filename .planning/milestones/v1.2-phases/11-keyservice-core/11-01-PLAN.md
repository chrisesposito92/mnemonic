```rust
/// Generates a cryptographically random API key token: "mnk_" + 64 hex chars.
fn generate_raw_token() -> String {
    let mut bytes = [0u8; 32];
    OsRng.try_fill_bytes(&mut bytes).expect("OsRng entropy unavailable");
    let hex: String = bytes.iter().map(|b| format!("{:02x}", b)).collect();
    format!("mnk_{}", hex)
}

/// Hashes a raw token string with BLAKE3, returning the 32-byte hash.
fn hash_token(raw: &str) -> blake3::Hash {
    blake3::hash(raw.as_bytes())
}
```

**Step 3: Implement create() method** (per D-01 through D-06, KEY-01)

Replace the `create` todo!() stub. Remove its `#[allow(dead_code)]` annotation. Implementation:
1. Call `generate_raw_token()` to get `raw_token` (e.g., "mnk_a1b2c3...")
2. Call `hash_token(&raw_token)` to get BLAKE3 hash
3. Derive `hashed_key = hash.to_hex().to_string()` (64-char hex string for DB storage)
4. Derive `display_id = hashed_key[..8].to_string()` (first 8 hex chars of hash — D-06, NOT raw token prefix)
5. Generate `id = uuid::Uuid::now_v7().to_string()`
6. INSERT into api_keys (id, name, display_id, hashed_key, agent_id) using `self.conn.call()`
7. Query back the inserted row to get server-generated `created_at`
8. Return `(ApiKey { id, name, display_id, agent_id, created_at, revoked_at: None }, raw_token)`

SQL for insert:
```sql
INSERT INTO api_keys (id, name, display_id, hashed_key, agent_id)
VALUES (?1, ?2, ?3, ?4, ?5)
```

SQL for query-back:
```sql
SELECT id, name, display_id, agent_id, created_at, revoked_at
FROM api_keys WHERE id = ?1
```

**Step 6: Implement validate() method** (per D-07, D-08, D-09, D-10, KEY-04, INFRA-02)

Replace the `validate` todo!() stub. Remove its `#[allow(dead_code)]` annotation. Implementation:
1. Hash incoming raw_token: `let incoming_hash = hash_token(raw_token);`
2. Get hex string: `let incoming_hex = incoming_hash.to_hex().to_string();`
3. Get incoming bytes: `let incoming_bytes: [u8; 32] = *incoming_hash.as_bytes();`
4. Use `self.conn.call()` with a single SQL query:
   ```sql
   SELECT id, hashed_key, agent_id FROM api_keys
   WHERE hashed_key = ?1 AND revoked_at IS NULL
   ```
5. If no row found (`QueryReturnedNoRows`), return error via `rusqlite::Error::QueryReturnedNoRows` which maps to `DbError::Query`
6. If row found, decode stored_hex to [u8; 32] using `blake3::Hash::from_hex(&stored_hex).unwrap().as_bytes()`, then call `constant_time_eq_32(&incoming_bytes, stored_bytes)`.
7. If match, return `Ok(AuthContext { key_id, allowed_agent_id: agent_id })`
8. If no match (hash collision in hex — extremely unlikely), return error

CRITICAL per INFRA-02: NEVER use `==` to compare hash values. Always use `constant_time_eq_32()` on `[u8; 32]`.
