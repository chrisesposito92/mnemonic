Insert the following SQL block inside the existing `c.execute_batch(...)` call, AFTER the `CREATE INDEX IF NOT EXISTS idx_compact_runs_agent_id ON compact_runs(agent_id);` line and BEFORE the closing `")?;` of execute_batch:

```sql
-- v1.2: API key authentication
CREATE TABLE IF NOT EXISTS api_keys (
    id          TEXT PRIMARY KEY,
    name        TEXT NOT NULL DEFAULT '',
    display_id  TEXT NOT NULL,
    hashed_key  TEXT NOT NULL UNIQUE,
    agent_id    TEXT,
    created_at  DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    revoked_at  DATETIME
);

CREATE INDEX IF NOT EXISTS idx_api_keys_hashed_key ON api_keys(hashed_key);
CREATE INDEX IF NOT EXISTS idx_api_keys_agent_id   ON api_keys(agent_id);
```

Column rationale per locked decisions:
- `display_id TEXT NOT NULL` — first 8 hex chars of BLAKE3(raw_key), NOT a prefix of the raw key (D-03)
- `hashed_key TEXT NOT NULL UNIQUE` — hex-encoded BLAKE3 hash, 64 chars (D-01)
- `agent_id TEXT` — NULL = wildcard, no sentinel strings (D-04)
- `revoked_at DATETIME` — NULL = active, soft delete (D-05)
- `CREATE TABLE IF NOT EXISTS` / `CREATE INDEX IF NOT EXISTS` — idempotent, safe on v1.1 databases (D-06)

Do NOT add `last_used_at` column (deferred to Phase 11 per research discretion resolution).

```rust
impl axum::response::IntoResponse for ApiError {
    fn into_response(self) -> axum::response::Response {
        let (status, body) = match &self {
            ApiError::BadRequest(msg) => (
                axum::http::StatusCode::BAD_REQUEST,
                serde_json::json!({"error": msg}),
            ),
            ApiError::NotFound => (
                axum::http::StatusCode::NOT_FOUND,
                serde_json::json!({"error": "not found"}),
            ),
            ApiError::Unauthorized(_) => (
                axum::http::StatusCode::UNAUTHORIZED,

            ),
            ApiError::Internal(e) => {

                (

                )
            }
        };
        (status, axum::Json(body)).into_response()
    }
}
```

- Do NOT add `Forbidden` variant — deferred to Phase 13 (D-10).
