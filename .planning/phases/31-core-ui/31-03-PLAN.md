**1. Create dashboard/src/components/Pagination.tsx** (per D-07; UI-SPEC Pagination contract):

Props:
```typescript
interface PaginationProps {
  offset: number
  limit: number
  total: number
  onPageChange: (newOffset: number) => void
}
```

Computed values:
- `start = offset + 1` (1-indexed display)
- `end = Math.min(offset + limit, total)`
- `hasPrev = offset > 0`
- `hasNext = offset + limit < total`

Layout:
- Container: display flex, alignItems center, justifyContent space-between, padding 8px 0
- Left: label text "Showing {start}--{end} of {total}" (exact Copywriting Contract), fontSize 12px, fontWeight 400, color var(--color-text-muted)
- Right: display flex, gap 8px
  - "Prev" button: fontSize 12px, fontWeight 400, padding 4px 12px, background var(--color-surface), border 1px solid var(--color-border), borderRadius 4px, color var(--color-text-muted), cursor pointer. Disabled (opacity 0.3, cursor default) when hasPrev=false. onClick: `onPageChange(offset - limit)`
  - "Next" button: same style. Disabled when hasNext=false. onClick: `onPageChange(offset + limit)`

Use en-dash character (--) in "Showing X--Y of Z" per UI-SPEC: the actual rendered character should be an en-dash (Unicode \u2013), matching the Copywriting Contract which says "Showing {start}\u2013{end} of {total}".

const uniqueAgents = [...new Set(data.memories.map(m => m.agent_id))]
