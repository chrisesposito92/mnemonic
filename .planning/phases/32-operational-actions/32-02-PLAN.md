    **State type (7 states per UI-SPEC):**

    ```typescript
    type CompactState =
      | { kind: 'idle' }
      | { kind: 'loading-dry-run' }
      | { kind: 'preview'; result: CompactResponse; memories: Map<string, Memory> }
      | { kind: 'loading-execute' }
      | { kind: 'success'; result: CompactResponse }
      | { kind: 'empty' }
      | { kind: 'error'; message: string }
    ```

    **Component state:**
    - `state: CompactState` -- starts as `{ kind: 'idle' }`
    - `agents: string[]` -- populated from fetchStats on mount

    - `threshold: string` -- default '0.85' (string for input control, parsed to float on submit)

    **On mount effect (agent dropdown population):**
    - Create AbortController
    - Set agentsLoading = true
    - Call `fetchStats(token, controller.signal)`
    - On success: set agents = data.agents.map(a => a.agent_id), agentsLoading = false
    - On UnauthorizedError: call onUnauthorized()
    - On AbortError: ignore
    - On other error: set agentsLoading = false (non-critical -- dropdown just won't populate)
    - Cleanup: controller.abort()
    - Dependencies: [token, onUnauthorized]

    **No agents empty state:**
    - If agentsLoading === false and agents.length === 0, render empty state:
      - Heading: "No agents available" (14px/400/--color-text)

      - Centered, padding 48px 0

      - Same surface/border style as select
      - onChange: update threshold state
    - "Run Dry Run" button:
      - Accent CTA style: padding 8px 16px, fontSize 14px, fontWeight 600, fontFamily inherit, background --color-accent, color --color-bg, border none, borderRadius 4px
      - Disabled (opacity 0.6, cursor default) when: selectedAgent === '' OR state.kind === 'loading-dry-run' OR state.kind === 'loading-execute'
      - onClick: trigger handleDryRun

    **handleDryRun function:**
    - Parse threshold: `const t = parseFloat(threshold)`. If isNaN(t) or t < 0 or t > 1, set state to error "Threshold must be a number between 0 and 1."
    - Set state to `{ kind: 'loading-dry-run' }`
    - Create AbortController (store in useRef for abort on unmount/discard)
    - Call `compactMemories(token, { agent_id: selectedAgent, threshold: t, dry_run: true }, controller.signal)`
    - On success:
      - If result.clusters_found === 0: set state to `{ kind: 'empty' }`
      - Else: collect all source_ids from result.id_mapping.flatMap(c => c.source_ids)
      - Fetch all source memories in parallel: `Promise.all(allSourceIds.map(id => fetchMemoryById(token, id, controller.signal).catch(() => null)))`
      - Build Map<string, Memory> from results (skip nulls from failed fetches)
      - Check controller.signal.aborted before setting state (prevent stale update -- Pitfall 3)
      - Set state to `{ kind: 'preview', result, memories: memoryMap }`
    - On UnauthorizedError: call onUnauthorized()
    - On AbortError: ignore (component unmounted or user discarded)
    - On other error: set state to `{ kind: 'error', message: 'Compaction failed. Check server logs and try again.' }`

    **handleExecute function:**
    - Set state to `{ kind: 'loading-execute' }`
    - Create AbortController
    - Call `compactMemories(token, { agent_id: selectedAgent, threshold: parseFloat(threshold), dry_run: false }, controller.signal)`
    - On success: set state to `{ kind: 'success', result }`
    - On UnauthorizedError: call onUnauthorized()
    - On AbortError: ignore
    - On error: set state to `{ kind: 'error', message: 'Compaction failed. Check server logs and try again.' }`

    **handleDiscard function:**
    - Abort the dry-run AbortController ref if active
    - Set state to `{ kind: 'idle' }` (preserve selectedAgent and threshold)

    **handleReset function (from success/error back to idle):**
    - Set state to `{ kind: 'idle' }`

    **Render by state.kind:**

    *idle:* Show controls row only (no results section)

    *loading-dry-run:* Show controls row (disabled) + `<SkeletonRows rows={5} />`

    *preview:* Show controls row (disabled) + summary line + cluster table + confirm/discard buttons
    - Summary line: "{result.clusters_found} clusters found, {result.memories_merged} memories -> {result.memories_created} compacted" at 14px/600/--color-text, marginBottom 16px
