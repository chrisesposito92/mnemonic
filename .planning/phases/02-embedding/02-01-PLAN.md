**d) LocalEngine::new() — model loading function:**

Must be a regular (non-async) function that:
1. Creates `hf_hub::api::sync::Api::new()` to download model files
2. Uses `Repo::with_revision("sentence-transformers/all-MiniLM-L6-v2", RepoType::Model, "refs/pr/21")` — this revision has safetensors weights
3. Downloads 3 files: `config.json`, `tokenizer.json`, `model.safetensors` via `repo.get()`
4. Parses config.json into `candle_transformers::models::bert::Config` via serde_json
5. Loads tokenizer via `Tokenizer::from_file(tokenizer_path)`
6. Creates `candle_core::Device::Cpu`
7. Loads weights via `unsafe { VarBuilder::from_mmaped_safetensors(&[weights_path], DTYPE, &device) }`
8. Creates BertModel via `BertModel::load(vb, &config)`
9. Wraps in `Arc<Mutex<LocalEngineInner>>` and returns `LocalEngine`
10. Maps ALL errors to `EmbeddingError::ModelLoad(e.to_string())`

**e) run_inference() — private function for single-text embedding:**

Signature: `fn run_inference(model: &BertModel, tokenizer: &tokenizers::Tokenizer, text: &str, device: &candle_core::Device) -> Result<Vec<f32>, EmbeddingError>`

Steps:
1. Encode text: `tokenizer.encode(text, true)` — the `true` adds special tokens ([CLS], [SEP])
2. Extract `token_ids` and `attention_mask` from the encoding (`.get_ids()`, `.get_attention_mask()`)
3. Build tensors with batch dim: `Tensor::new(ids, device)?.unsqueeze(0)?` for both token_ids and attention_mask
4. Create `token_type_ids = token_ids.zeros_like()?`
5. Run forward: `model.forward(&token_ids, &token_type_ids, Some(&attention_mask))?` — output shape (1, n_tokens, 384)
6. Apply mean pooling with attention mask (see step f below)
7. Assert result length is 384: if not, return `EmbeddingError::Inference(format!("expected 384 dimensions, got {}", result.len()))`
8. Return `Vec<f32>`

**f) mean_pool_and_normalize() — private function:**

CRITICAL: Use the exact tensor operations from the verified candle bert/main.rs example:
```rust
fn mean_pool_and_normalize(
    embeddings: &candle_core::Tensor,    // (1, n_tokens, 384)
    attention_mask: &candle_core::Tensor, // (1, n_tokens)
) -> Result<candle_core::Tensor, candle_core::Error> {
    use candle_transformers::models::bert::DTYPE;
    let attention_mask_f = attention_mask.to_dtype(DTYPE)?.unsqueeze(2)?; // (1, n_tokens, 1)
    let sum_mask = attention_mask_f.sum(1)?;                               // (1, 1)
    let pooled = (embeddings.broadcast_mul(&attention_mask_f)?).sum(1)?;   // (1, 384)
    let pooled = pooled.broadcast_div(&sum_mask)?;                         // (1, 384)
    // L2 normalize
    let norm = pooled.sqr()?.sum_keepdim(1)?.sqrt()?;
    Ok(pooled.broadcast_div(&norm)?)
}
```

Do NOT use CLS token pooling. Do NOT use simple mean without attention mask. These are explicitly wrong for all-MiniLM-L6-v2.

**g) EmbeddingEngine impl for LocalEngine:**

```rust
#[async_trait]
impl EmbeddingEngine for LocalEngine {
    async fn embed(&self, text: &str) -> Result<Vec<f32>, EmbeddingError> {
        if text.is_empty() {
            return Err(EmbeddingError::EmptyInput);
        }
        let inner = std::sync::Arc::clone(&self.inner);
        let text = text.to_string();
        tokio::task::spawn_blocking(move || {
            let guard = inner.lock().map_err(|_| EmbeddingError::Inference("mutex poisoned".into()))?;
            run_inference(&guard.model, &guard.tokenizer, &text, &guard.device)
        })
        .await
        .map_err(|e| EmbeddingError::Inference(e.to_string()))?
    }
}
```
