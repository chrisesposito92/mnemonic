### Key Integration Findings

1. **init helpers correctly scoped:** `init_db()` (DB-only, Phase 16) used by Keys + Recall; `init_db_and_embedding()` (Phase 17) used by Remember + Search; `init_compaction()` (Phase 19) used by Compact only. Ownership semantics enforced by Rust compiler.

2. **--json flag properly threaded:** Global `--json` on Cli struct extracted before match in main.rs, passed to all 5 dispatch arms, consumed by all 8 data-producing handlers.

3. **Commands enum exhaustive:** 6 variants (Serve, Keys, Recall, Remember, Search, Compact) + None fallthrough. Rust match enforces exhaustiveness at compile time.

4. **stdin isolation:** `IsTerminal` + `read_to_string` path exists only in the Remember dispatch arm.

5. **Full lifecycle wired:** remember -> recall --id -> search -> compact -> recall all chained correctly through shared DB.
