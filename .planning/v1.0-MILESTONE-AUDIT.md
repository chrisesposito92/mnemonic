### Cross-Phase Wiring

| Connection | Status | Notes |

### E2E Flows (2/2 complete)

1. **Store and search** — POST /memories -> embed -> atomic dual-table insert -> GET /memories/search -> embed query -> CTE KNN -> ranked results
2. **Multi-agent isolation** — Two agents store memories -> search scoped by agent_id -> each sees only their own

### Integration Issues

**INT-01 (Low): mnemonic.toml.example missing openai_api_key**
- `openai_api_key` added to Config in Phase 2 but not backported to `.example` file
- **Mitigated:** README correctly documents `MNEMONIC_OPENAI_API_KEY`
- Affected: EMBD-04, CONF-03

**INT-02 (Medium): embedding_provider config field is dead**
- `main.rs` selects engine based solely on `config.openai_api_key.is_some()`
- `config.embedding_provider` is logged at startup but never checked
- Setting `MNEMONIC_EMBEDDING_PROVIDER=openai` without API key silently uses local engine
- **Mitigated:** Runtime behavior is safe; the correct engine always loads
- Affected: CONF-02, EMBD-04
